# Classificador - Placas de Trânsito

Projeto com o intuito de criar um classificador que identifica placas de trânsito. Apesar de ser simples, aborda diversos conceitos interessantes, como: separação de dados para treino, teste e validação; exibição de gráficos; processamento de imagens; data augmentation; criação e treinamento de CNN; validação dos resultados.

A instalação das dependências necessárias para o projeto pode ser feita através do comando: 

`pip install -r requirements.txt`
